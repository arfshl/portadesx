### List of portadesx-commands
##### Termux Shell
- `portadesx-cli` - for enter to proot shell
- `portadesx-gui` - start X11 session
- `portadesx-help` - print help message
##### PRoot shell
- `startvnc` - start vnc server
- `stopvnc` - stop vnc server
